# Système de gestion académique - C++
## Description
Ce projet est un système de gestion académique développé en C++ en utilisant la programmation orientée objet (POO).  
Il permet de gérer les étudiants, les modules, les inscriptions et les évaluations.

Le programme calcule automatiquement :
- les moyennes des modules
- la moyenne générale des étudiants

Le système permet également l’enregistrement des données et leur chargement automatique au démarrage du programme.

## Fonctionnalités
- Ajouter un étudiant
- Ajouter un module
- Inscrire un étudiant à un module
- Ajouter des contrôles et des notes
- Calculer la moyenne d’un module
- Calculer la moyenne générale
- Afficher les étudiants et les modules

## Technologies utilisées
- C++
- Programmation orientée objet (POO)
